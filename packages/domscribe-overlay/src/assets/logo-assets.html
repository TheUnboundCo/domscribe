1. Wordmark Specification

The wordmark is rendered as styled text, not SVG. Use these specs for implementation.

Property Value
Font Family 'Inter', system-ui, sans-serif
Font Weight 600 (semi-bold)
Letter Spacing -0.02em
Text Transform lowercase
Color (Dark BG) #fafafa
Color (Light BG) #18181b
